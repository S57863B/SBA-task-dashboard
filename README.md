# TaskMaster - Task Management Dashboard

## Overview
Task Management Dashboard built with React and TypeScript. It features a modern user interface that allows users to create, read, update, delete, and filter tasks efficiently.

## Features
- **CRUD Operations**: Add, view, edit (status), and delete tasks.
- **Advanced Filtering**: Filter tasks by status, priority, and search queries.
- **Persistent Storage**: Tasks are automatically saved to your browser's `localStorage`.
- **Responsive Design**: Beautiful UI built with Tailwind CSS, supporting both mobile and desktop layouts.
- **Dark Mode**: Built-in toggle for light and dark themes.

## Tech Stack
- **React**: Functional components and Hooks (`useState`, `useEffect`)
- **TypeScript**: Strict typing with custom Interfaces and Types
- **Vite**: Next-generation frontend tooling and build execution
- **Tailwind CSS**: Utility-first CSS framework for styling
